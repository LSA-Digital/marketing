# HeyGen Script: Community-benefit planning should start where need is highest

## Production Notes
- **Speaker:** Dr. Thad Perry
- **Format:** direct-to-camera
- **Target length:** 45-60 seconds
- **Tone:** educational, credible, non-salesy
- **Audience:** business; hospital community-benefit leaders and CHNA planners
- **Objective:** build trust and prepare outreach by showing why tract- and county-level prioritization creates a more defensible starting point for community-benefit decisions
- **Desired outcome:** viewers understand that broad averages are not enough when deciding where community health investment should go first
- **Primary anchor:** Tennessee-oriented HSRA tract/county prioritization framing using an auditable public-data baseline

## Script
When hospitals plan community-benefit work, one of the hardest questions is where to start. In my experience, county averages can be useful, but they often smooth over the neighborhoods carrying the highest burden.

What we have been building in Tennessee starts with an auditable public-data baseline for health burden, environmental burden, and social conditions. Then we use that view to see which counties and tracts are carrying the most pressure.

That matters because community-benefit planning becomes more defensible when resources are tied to actual local conditions instead of broad assumptions. It helps teams make a clearer case for why screening, care navigation, outreach, or other interventions should go to one place before another.

The main point is simple: if you want measurable impact, start with a clearer picture of where need is highest.

## On-Screen Text Cues
- County averages can hide neighborhood-level need
- Start with an auditable public-data baseline
- Prioritize counties and tracts with the highest burden
