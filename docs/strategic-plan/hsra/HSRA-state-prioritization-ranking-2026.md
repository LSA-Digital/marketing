# HSRA State Prioritization Ranking 2026

**Last Updated:** 2026-03-10

## Purpose

This document replaces the earlier loose state-targeting logic with a more defensible rank-ordering model for HSRA / PopHealthMap outreach. It uses per-state public evidence where available, separates `problem fit` from `messaging responsiveness`, and avoids pretending we can predict exact reply rates from public signals alone. It now also incorporates burden-led differentiators surfaced in the deeper state framework review, especially rural hospital vulnerability, cancer-screening disparity pressure, and Medicaid non-expansion urgency.

## Important Interpretation Rule

This ranking is **not** saying lower-ranked states do not matter. It is saying:

- all 50 states should still receive launch messaging
- the top-ranked states deserve the fastest research, strongest state-specific message tailoring, and earliest live discovery effort
- lower-ranked states should stay in launch coverage and nurture until new evidence or real campaign responses change the picture

## Methodology

### Two-axis model

Following Oracle review, each state is assessed on two separate dimensions:

1. `Problem fit` - does the state show visible public evidence of the kind of planning, screening, disparity, or prioritization problem HSRA solves?
2. `Messaging responsiveness` - does the state show public evidence that HSRA-style messaging will look familiar, timely, and actionable to likely buyers?

### Scoring rubric

Each sub-signal is scored coarsely to avoid fake precision.

| Score | Meaning |
| --- | --- |
| 0 | no meaningful differentiating public signal found in the current sweep |
| 1 | weak or indirect signal |
| 2 | one strong public signal or multiple moderate signals |
| 3 | multiple strong and recent public signals |

### Problem fit components

| Component | Weight | Evidence used |
| --- | ---: | --- |
| Rural burden and coverage pressure | 25% | rural hospital vulnerability, rural outcome burden, Medicaid non-expansion pressure |
| Public-health planning pressure | 20% | SHA, SHIP, CHA, CHIP, PHAB, CHNA, county indicator infrastructure |
| EJ / environmental-health screening need | 20% | official state EnviroScreen-style tools or visible tract-level EJ logic |
| Cancer / screening / disparity targeting fit | 20% | public cancer-control, screening, or disparity-targeting signal |
| RHT intensity | 15% | FY2026 CMS Rural Health Transformation award amount |

### Messaging responsiveness components

| Component | Weight | Evidence used |
| --- | ---: | --- |
| State dashboard / analytics maturity | 35% | official state dashboards, public indicator portals, health equity tools |
| Adjacent procurement / configured analytics evidence | 35% | public dashboard or analytics engagements, CHNA/CHA collaboration evidence |
| Message-lane clarity | 20% | whether a clear state-specific message lane is obvious from public evidence |
| Access-path clarity | 10% | whether likely account owners are easy to identify from public structures |

### RHT intensity buckets

Because all 50 states received RHT awards, this is a differentiator only by degree.

| FY26 award amount | RHT intensity score |
| --- | ---: |
| `>= $220M` | 3 |
| `$200M-$219.9M` | 2 |
| `$180M-$199.9M` | 1 |
| `< $180M` | 0 |

### Confidence labels

| Confidence | Meaning |
| --- | --- |
| High | 3 or more recent sources across multiple signal types |
| Medium | 2 useful sources or mixed evidence strength |
| Low | mostly baseline RHT signal or limited state-specific evidence |

## Evidence Base Used

This ranking uses the following evidence categories already gathered in repo research plus targeted state-source checks:

- official CMS FY2026 RHT award amounts for all 50 states
- rural hospital vulnerability and rural burden signals from Chartis and related public-health reference sources
- official or public state EJ / burden-screening tools: California, Colorado, Connecticut, Illinois, Maryland, Minnesota, New Jersey, Washington, and related state materials for New York
- official state health-equity or disparity dashboards: Connecticut, Kentucky, Pennsylvania, New York, Minnesota, Washington
- official SHA / SHIP / county-indicator or state assessment infrastructure: Pennsylvania, Oregon, Minnesota, Nebraska, New York
- public CHNA / CHA or analytics-procurement analogs: North Carolina, Connecticut, Kentucky, New York
- public cancer-control / screening-gap and Medicaid-expansion-pressure evidence from the deeper state framework review
- internal commercial anchor evidence for Tennessee

## Why Some States Rank Above Others

### The strongest differentiators found so far

1. `Rural burden, hospital vulnerability, and coverage-gap pressure`
   - strongest examples: Mississippi, Arkansas, Alabama, Kansas, Tennessee, Texas, Georgia
2. `Official state EJ / burden-screening tools`
   - strongest examples: California, Colorado, Connecticut, Illinois, Maryland, Minnesota, New Jersey, Washington
3. `Official state health-equity dashboards or disparity data tools`
   - strongest examples: Connecticut, Kentucky, Pennsylvania, New York, Minnesota, Washington
4. `Official SHA / SHIP / county indicator infrastructure`
   - strongest examples: Pennsylvania, Oregon, Minnesota, Nebraska, New York
5. `Explicit public analytics / CHNA / assessment analogs`
   - strongest examples: North Carolina, Connecticut, Kentucky, New York
6. `Cancer-control and screening-gap urgency`
   - strongest examples: Maryland, Georgia, Mississippi, Kentucky, New Mexico
7. `Exceptional RHT funding intensity`
   - strongest examples: Texas, Alaska, California, Montana, Oklahoma, Kansas

### Why Texas and Colorado score especially well

- `Texas` combines the strongest RHT award amount with an explicit public research analog around county-level outcome prediction and strong hospital planning evidence.
- `Colorado` has unusually clear official evidence of tract-level environmental burden scoring already being used to direct funding and resources.

### Why Pennsylvania and Connecticut moved up

- `Pennsylvania` has both official health-equity analysis tools and official SHA dashboard infrastructure, which raises both problem fit and message receptivity.
- `Connecticut` has public health-equity dashboards plus an official EJ screening tool, which makes HSRA easier to explain as a configured, state-first analytics layer.

### Why Maryland, Georgia, Mississippi, and Vermont now matter more

- `Maryland` now looks stronger because it combines a state EJ analog, a current rural-health strategic plan, and an active cancer-control planning lane.
- `Georgia` gains importance because non-expansion pressure, rural burden, and a recent cancer-control plan create a real unmet-need story even if the buyer path is not yet as clear.
- `Mississippi` remains a burden-led challenger with unusually strong hospital-vulnerability and disparity signals; it is now hard to justify keeping it out of the fast-follower set.
- `Vermont` has a smaller market surface area, but its extreme rurality plus recent health-equity and EJ-tool work make it stronger than a generic small-state assumption would suggest.

## Recommended Use In Campaign Planning

- `Launch to all 50 states` remains the right top-of-funnel move.
- `Prioritize Tier A first` for research depth, Thad-led first-touch customization, and earliest Mike discovery participation.
- `Use Tier B as the fast-follower bench` for message-lane testing and second-wave discovery.
- `Keep Tier C in broad launch coverage and nurture` until either public evidence improves or real campaign behavior proves stronger receptivity than the current public record suggests.

## What Still Needs More Research

The current ranking is materially better than the earlier one, but it is still limited by uneven public evidence across states. The biggest remaining gap is not national demand; it is `state-specific differentiation evidence` for middle-tier states.

Highest-value next research moves:

1. deepen the per-state review for Tier B and upper Tier C states with strong RHT awards but thin public-health analytics evidence
2. add a `commercial access path` overlay based on actual named contacts, partner routes, and existing LSA relationships
3. test campaign response data against this ranking and revise quarterly
